# AN-EXPLAINABLE-DEEP-NEURAL-NETWORK-APPROACH-FOR-NON-HODGKIN-LYMPHOMA-CLASSIFICATION
An explainable deep learning framework for classifying Non-Hodgkin Lymphoma subtypes (MCL, FL, CLL) from histopathological images. Integrates Grad-CAM and SHAP to visualize model decisions, improving transparency and trust in medical image classification systems.
